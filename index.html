About

After more than a decade working hands-on with technical systems in the field, I transitioned into software development, bringing with me a strong foundation in troubleshooting, system reliability, and solving real-world problems. Today, I build modern applications using Python, Flask, FastAPI, SQL/ORM, HTML, and CSS, and I enjoy creating clean, practical solutions with tools like GitHub and SQLite. Along the way, I’ve developed a strong interest in Artificial Intelligence, especially Generative AI, working with tools like LangChain and retrieval-augmented generation (RAG), as well as prompt engineering and automation workflows using n8n. I’m still early in my journey as a software engineer, but I’m genuinely curious and always eager to learn. I enjoy exploring new technologies, building AI-driven applications, and turning ideas into real, useful products. Recently, I’ve been focusing more on NLP and large language models to create intelligent, scalable features that combine traditional software engineering with modern AI capabilities. For me, every project is an opportunity to grow, improve, and understand things more deeply—and I’m excited to keep learning and building along the way.
